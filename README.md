# chatbot-v15

> Multi-version React + TypeScript chatbot UI playground

A scratchpad repository that tracks the iterative evolution of a React/TypeScript chatbot UI across four versions: v15 (baseline), v27 (document attachments), v28 (document attachments + theme switcher), and v41 (attachments + login + settings). Each folder is a standalone build with progressive feature additions.

🔗 **Live repo:** <https://github.com/girishlade111/chatbot-v15>

## ✨ Features

- Four progressive chatbot builds you can compare side-by-side
- Document attachment support (v27 → v41)
- Light/dark theme toggle (v28 onwards)
- Login + settings panels (v41)
- Standalone folders — open any version independently

## 🛠️ Tech stack

React 18 • TypeScript • Lucide React icons • Tailwind CSS • shadcn/ui

## 🚀 Getting started

```bash
cd "chatbot v41 with attachments, login"   # or any version folder
npm install
npm run dev
```
